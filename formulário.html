Botões

Opções

classe descrição
.button Classe base para todos os botões.
.button-disabled Botão desabilitado
.button-default Botão com layout padrão.
.button-success Botão que conclui alguma ação.
.button-danger Botão que indica situação de perigo, normalmente ações irreversíveis.

Padrão

Link 
<button class="button button-default">Button</button>
<a href="#" class="button button-default">Link</a>

Desabilitado

Link 
<button class="button button-disabled" disabled>Button</button>
<a href="#" class="button button-disabled">Link</a>

Sucesso

Link 
<button class="button button-success">Button</button>
<a href="#" class="button button-success">Link</a>

Excluir

Link 
<button class="button button-danger">Button</button>
<a href="#" class="button button-danger">Link</a>

Grid

O form é um grid comum para se implementar os campos.

Estrutura

<form class="form">
  <div class="field">(campo 1)</div>
  <div class="field">(campo 2)</div>
  <div class="field">(campo 3)</div>
  <div class="field">(...)</div>
</form>

Exemplo

<form class="form">
  <div class="field">
    <label>Texto:</label>
    <input type="text" class="input" value="Texto" />
  </div>

  <div class="field">
    <label>Checkbox:</label>

    <label class="checkbox">
      <input type="checkbox"> <span>A</span>
    </label>

    <label class="checkbox">
      <input type="checkbox"> <span>B</span>
    </label>

    <label class="checkbox">
      <input type="checkbox"> <span>C</span>
    </label>
  </div>

  <div class="field">
    <label>Radio:</label>

    <label class="radio">
      <input type="radio" name="opcao" value="a" checked> <span>Opção A</span>
    </label>

    <label class="radio">
      <input type="radio" name="opcao" value="b"> <span>Opção B</span>
    </label>

    <label class="radio">
      <input type="radio" name="opcao" value="c"> <span>Opção C</span>
    </label>
  </div>

  <div class="field">
    <label>Select:</label>
    <select class="combo">
      <option>Opção 1</option>
      <option>Opção 2</option>
      <option>Opção 3</option>
    </select>
  </div>
</form>

Senha

<input type="password" value="Texto" class="input" disabled/>

<input type="password" value="Texto" class="input input-error" />
