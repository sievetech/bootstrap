Box

O box é um grid comum para se implementar funcionalidades autosuficientes.

Estrutura

<article class="box">
  <header>(opcional)</header>

  <section>...</section>

  <footer>(opcional)</footer>
</article>

Grid

O grid é como uma estante que acomoda os componentes. Com ele você pode posicioná-los facilmente sem trabalhar com css. Para isso, bastanta criar um <div class="container">, colocar o número de linhas <div class="row"> e definir as colunas com as classes .col-md-* onde * varia de 1 a 12. Ótimo para páginas no tipo .page-basic. Por ora utilizamos o grid do twitter, veja a documentação completa aqui.

Pages

O page é um grid comum para se implementar páginas. Esse é o primeiro passo na construção de uma página e entra direto no <body>.

Estrutura

<main class="page">
  <header>(opcional)</header>

  <section>...</section>

  <footer>(opcional)</footer>
</article>

Opções

classe descrição
.page Classe base para todas as páginas.
.page-basic Tipo mais básico de página. Composto por cabeçalho, conteúdo e rodapé.
.page-fixed-header Utilizado junto com .page-basic, faz com que o header fique fixo no scroll.
.page-fixed-footer Utilizado junto com .page-basic, faz com que o footer fique fixo no scroll.
.page-modal Página que possue um modal centralizado. Ótimo para telas de login, alteração de senha e warnings.
